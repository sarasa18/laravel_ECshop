## インストール後の実施事項

画像のダミーデータは
public/imagesフォルダ内に
sample1.jpg ~ sample6.jpg として保存しています。

php artisan storage:link でstorageフォルダにリンク後、
storage/app/public/productsフォルダ内に保存すると表示されます。
（productsフォルダがない場合は作成してください。）

ショップの画像も表示する場合は、
storage/app/public/shopsフォルダを作成し,その中に保存すると表示されます。
